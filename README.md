# lfspkg

`lfspkg` 是一个偏向 **LFS / BLFS / 自建系统** 的极简包跟踪器。

它的目标不是替代 `apt` / `dnf` / `pacman` 这种发行版级包管理器，而是解决 LFS 用户最痛的几个问题：

- 我到底装了哪些文件？
- 这个文件是谁装的？
- 卸载一个软件时，怎么尽量删干净？
- 一个包依赖哪些我自己编出来的包？
- 能不能先把东西装进 staging 目录，再统一登记安装？

这个项目的设计重点是：

1. **文件级跟踪**：记录每个包装了哪些文件、软链接和目录。
2. **依赖关系**：支持显式依赖、反向依赖检查、依赖树显示。
3. **干净卸载**：按安装清单删除文件，并尝试删除空目录。
4. **分离安装**：推荐先 `DESTDIR=/tmp/pkg/... install`，再用 `lfspkg` 从 staging 目录复制到目标根目录并登记。
5. **LFS 友好**：单文件 C++17，无第三方库依赖。
